    KK
    2022-08-07

    写在前面

    关于命令行

    我们常说的命令行其实指的是Shell。Shell是一个程序，它接收由键盘输入的命令并传递给操作系统来执行。几乎所有Linux发行版都提供了来自GNU项目的Shell程序Bash。其中，Bash是Bourne Again Shell的缩写，以此表明是sh的增强版。大多数命令行形式如下所示：

    • options：表示修改命令行为的选项
    • arguments：表示 一个或多个参数

    关于一些表示

    • file表示文件，directory表示目录，item表示file或directory
    • 参数后面出现三个点号，表示该参数可以重复出现：比如 mkdir directory... 表示该命令可以采用 mkdir dir1 dir2 dir3 的形式

    导航

    文件名

    在Linux系统中：

    • 以点号开头的为隐藏文件
    • 区分大小写
    • 没有”文件扩展名“的概念

    pwd 输出当前工作目录名

    ls 列出目录内容

    通过给ls命令添加-l选项得到长格式的输出结果：

    以第一行为例，长格式各字段含义：

    字段 含义
    -rw-r–r–@ 文件访问权限。
    1 硬链接数量
    sibyl 文件属主
    staff 文件属组

    ls命令的常用选项：

    选项 含义
    -a 列出所有文件（包括隐藏文件）
    -h 以 human-readable 形式显示文件大小（不再以字节 B 为单位）
    -l 以长格式显示
    -r 以降序显示（默认按字母升序显示）
    -S 按文件大小排序
    -t 按修改日期排序

    file 确定文件类型

    操作文件和目录

    通配符

    常用通配符

    通配符 含义
    * 匹配任意多个字符
    ？ 匹配任意单个字符
    [characters] 匹配属于字符集合中的任意单个字符
    [!characters] 匹配不属于字符集合中的任意单个字符
    [[:class:]] 匹配属于字符类class中的任意单个字符

    常用字符类

    字符类 含义
    [:digit:] 匹配任意单个数字
    [:lower:] 匹配任意单个小写字母
    [:upper:] 匹配任意单个大写字母

    通配符示例

    模式 匹配
    b*.txt 以b开头扩展名为.txt的文件
    Data??? 以Data开头并紧接3个字符的文件
    [abc]* 以a、b、c中任意字符开头的文件
    BACKUP[0-9][0-9][0-9] 以BACKUP开头并紧接3个数字的文件
    [![:digit:]]* 不以数字开头的文件
    *[[:lower:]123] 以小写字母或1、2、3中任意数字结尾的文件

    mkdir 创建目录

    cp 复制文件和目录

    cp命令的常用选项：

    选项 含义
    -i 在覆盖已有文件之前，提示用户确认
    -r 递归复制（在复制目录时用到）
    -u 只复制目标目录中不存在或者比目标目录更新的文件
