Linux命令行--基础知识

Sibyl K
2022-08-07

[TOC]

大多数命令行形式：

  • -options 修改命令行为的选项
  • arguments 一个或多个参数

导航

pwd：输出当前工作目录名称
ls： 列出目录内容

ls 命令常用选项

选项 描述
-a 列出所有文件（包括隐藏文件）
-h 以 human-readable 形式显示文件大小（不再以字节 B 为单位）
-l 以长格式显示
-r 以降序显示（默认按字母升序显示）
-S 按文件大小排序
-t 按修改日期排序

长格式各字段含义

字段 含义
-rw-r-r– 文件访问权限。
1 硬链接数量
root 文件属主
root 文件属组
cd：修改目录
mkdir：创建目录
cp：复制文件和目录
mv：移动/重命名文件和 目录
rm：删除文件和目录
ln：创建硬链接/符号链接
