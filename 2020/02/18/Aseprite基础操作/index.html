Aseprite基础操作

  星期二, 二月 18日 2020, 10:23 上午

Aseprite是一个简单易上手的2D像素画工具：官网下载 | Steam下载 | 推荐教程

Basic Tools

Pen Tool [B] : pixel-perfect | symmetry option | ink option

Line Tool [L] : shift+L(特殊角度) | ctrl+L(中心点)

Shape Tool [U]：shift+L(正方正圆) | ctrl+L(中心点)

Contour [D] : 填充

Eraser [E] : 擦除

Eyedropper [ALT] : 取色

Paint Bucket [G] : continuious | 配合Selection Tool

Selection Tool [M] or [W] : continuious | 包含魔棒 | ctrl+D(取消选区)

Layers

Press [TAP]

Color Tool

Palette Editor [F4]

Sorting Palette : 比如创建两种颜色之间的色板并排序by gradient

Presets ：预设调色板

Options : Create palette from the current seprite
